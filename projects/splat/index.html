SPLAT

Novel single-cell copy number calling method for tagmentation-based scDNA-seq

SPLAT (Single-cell PLoid Analysis Tool) is a novel computational method for calling copy number variants (CNVs) from tagmentation-based single-cell DNA sequencing data, developed in the Van Loo Lab at MD Anderson Cancer Center.

Motivation

Intratumoral heterogeneity (ITH) — the coexistence of genetically distinct subclones within a single tumor — is a major driver of cancer evolution, treatment resistance, and metastasis. Resolving subclonal copy number architecture at single-cell resolution is essential for understanding clonal dynamics, yet existing methods were not designed for the sparse, low-coverage profiles generated by tagmentation-based protocols.

Method Overview

SPLAT uses a per-bin signal matrix (read counts, overlap counts, insert size distributions) derived from BAM files via the companion processBam R/Rcpp package. It constructs an S3 splat object with bins × cells matrices and applies statistical models to call integer copy number states across the genome, accounting for technical noise, GC bias, and mappability.

Key Features

  • BAM-level signal extraction via processBam (R/Rcpp wrapping htslib)
  • HPC-compatible batch processing using mclapply and LSF job arrays
  • Tidyverse-native interface for downstream analysis and visualization
  • Handles sparse data from tagmentation protocols (e.g., 10x Chromium CNV)

Status

Actively under development. Manuscript in preparation.
